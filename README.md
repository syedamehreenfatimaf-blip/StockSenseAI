# 📊 StockSense AI

An AI-Powered Inventory & Sales Dashboard built using modern web technologies.

## ✨ Features

- 📦 Inventory Management
- 💰 Sales Management
- 👥 Customer Management
- 🚚 Supplier Management
- 📊 Analytics Dashboard
- 🤖 AI Assistant
- 📈 Reports
- 🔔 Notifications
- ⚙️ Settings Panel
- 🌙 Dark Mode
- 📱 Responsive UI

## 🛠 Tech Stack

- React
- TypeScript
- TanStack Router
- Vite
- Tailwind CSS
- Shadcn UI
- Bun / npm

## Installation

```bash
git clone https://github.com/syedamehreenfatimaf-blip/StockSenseAI

cd StockSenseAI

npm install

npm run dev
```
